<!-- 20-Description -->    
Globale Beschreibung: **Nicht alle öffentlichen Dienstleistungen werden von allen benötigt oder können von allen genutzt werden. So wird beispielsweise der von den europäischen Ländern betriebene Visadienst nicht von den europäischen Bürgern benötigt, wohl aber von einigen Bürgern aus anderen Ländern, oder die öffentlichen Dienste, die Arbeitslosenunterstützung und Zuschüsse anbieten, sind auf bestimmte gesellschaftliche Gruppen ausgerichtet. Das CPSV verwendet das Core Criterion and Core Evidence Vocabulary (CCCEV - https://joinup.ec.europa.eu/release/core-criterion-and-core-evidence-vocabulary-v100) für diese Klasse. Das CCCEV enthält mehr Details, aber die Klasse Criterion Requirement hat drei obligatorische Eigenschaften**.    
Version: 0.0.1
